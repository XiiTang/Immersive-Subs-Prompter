# Chrome Web Store 商品信息

## 名称
Immersive Subs Prompter

## 简短描述 (132字符以内)
与桌面应用配合，捕获视频网站播放信息，实现沉浸式字幕学习体验。

English: Captures video playback info and syncs with desktop app for immersive subtitle learning.

## 详细描述

### 中文版本

🎯 **Immersive Subs Prompter** 是一款沉浸式字幕学习工具的浏览器扩展组件。

**✨ 主要功能：**
• 自动检测主流视频网站的播放状态
• 实时同步视频播放进度到桌面应用
• 支持字幕下载和显示
• A-B循环复读功能
• 单句循环学习

**🎬 支持的视频网站：**
• YouTube
• Bilibili
• Douyin
• Jellyfin / Emby（需配合桌面应用和媒体服务器配置）

**🔧 使用方法：**
1. 安装此浏览器扩展
2. 下载并运行 Immersive Subs Prompter 桌面应用
3. 打开任意支持的视频网站开始学习

**🔒 隐私保护：**
• 不收集个人身份信息
• 扩展设置保存在浏览器本地
• 播放和页面元数据仅发送到你配置的桌面应用 WebSocket 地址
• 如启用 Whisper-compatible API、媒体服务器或字幕下载，相关请求由桌面应用按你的配置发起

### English Version

🎯 **Immersive Subs Prompter** is a browser extension for immersive subtitle learning.

**✨ Key Features:**
• Automatically detects video playback on major video sites
• Real-time sync with desktop application
• Subtitle download and display
• A-B loop for repeated practice
• Single sentence loop learning

**🎬 Supported Video Sites:**
• YouTube
• Bilibili
• Douyin
• Jellyfin / Emby with the desktop app and media server configuration

**🔧 How to Use:**
1. Install this browser extension
2. Download and run the Immersive Subs Prompter desktop app
3. Open any supported video site to start learning

**🔒 Privacy:**
• No personal identity data collection
• Extension settings are stored locally in your browser
• Playback and page metadata is sent only to desktop app WebSocket endpoints you configure
• If enabled, Whisper-compatible APIs, media servers, or subtitle downloads are contacted by the desktop app according to your settings
