# Chrome Web Store 商品信息

## 名称
Immersive Subs Prompter

## 简短描述 (132字符以内)
与桌面应用配合，捕获视频网站播放信息，实现沉浸式字幕学习体验。

English: Captures video playback info and syncs with desktop app for immersive subtitle learning.

### English Version

🎯 **Immersive Subs Prompter** is a browser extension for immersive subtitle learning.

**✨ Key Features:**
• Automatically detects video playback on major video sites
• Real-time sync with desktop application
• Subtitle download and display
• A-B loop for repeated practice
• Single sentence loop learning

**🎬 Supported Video Sites:**
• YouTube
• Netflix
• Bilibili
• And more...

**🔧 How to Use:**
1. Install this browser extension
2. Download and run the Immersive Subs Prompter desktop app
3. Open any supported video site to start learning

**🔒 Privacy:**
• All data processed locally
• No personal data collection
• No external server communication
• Only communicates with local desktop app
