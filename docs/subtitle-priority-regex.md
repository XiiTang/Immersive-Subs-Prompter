# 字幕优先级正则示例

本页展示如何用正则表达式匹配完整字幕文件名（含后缀），并提供常用模板。默认大小写敏感，如需忽略大小写请在模式中自行声明。

## 快速示例

| 场景 | 正则 | 说明 |
| --- | --- | --- |
| 精确匹配英文内嵌字幕 | `\\.en\\.` | 仅匹配文件名中带 `.en.` 的字幕，如 `movie.en.srt` |
| 匹配 AI 生成的英文字幕 | `ai-en` | 适配包含 `ai-en` 的 AI 字幕标记 |
| 同时匹配简体/繁体中文 | `zh-Hans|zh-Hant|zh` | 使用 `|` 组合多种语言标记 |
| 优先日语字幕 | `\\.ja\\.|jpn` | 兼容 `.ja.` 与 `jpn` |
| 过滤特定版本 | `WEB-DL` | 仅选取文件名包含 `WEB-DL` 的字幕 |

## 高级用法

- **忽略大小写**：在模式前添加 `(?i)`，如 `(?i)english`。
- **排除噪声**：用负向前瞻排除关键词，例如 `^(?!.*(commentary|hearing-impaired)).*\\.en\\.srt$`。
- **多语言兜底**：`(en|eng|english)`、`(cmn|zh-Hans|zh_CN)`，将常见缩写放在一组内。
- **限定扩展名**：`\\.srt$` 或 `\\.ass$`，确保只匹配指定格式。

## 常见配置模板

### 英语学习者（英文主字幕，中文辅助）

- 主字幕：`(?i)en`, `ai-en`, `english`
- 副字幕：`(?i)zh-Hans`, `ai-zh`, `cmn`, `zh`

### 日语学习者（中文字幕主，日文辅助）

- 主字幕：`(?i)zh-Hans`, `zh`, `cmn`
- 副字幕：`(?i)ja`, `jpn`, `jp`, `\\.(ja|jpn)\\.`

### 排除机器翻译，优先官方字幕

- 主字幕：`(?i)en`, `english`, `\\.(en|eng)\\.`
- 副字幕：`(?i)zh-Hans`, `zh`
- 备用：`^(?!.*ai-).*(en|zh).*$`（跳过文件名包含 `ai-` 的字幕）

## 注意事项

- 特殊字符（如 `.`、`?`、`+`、`*`）需用反斜杠转义，否则将被当作正则元字符。
- 模式作用于完整文件名（例如 `video.ai-en.srt`），请按需包含扩展名或分隔符。
- 无效正则会在匹配时被跳过并记录警告，可在设置界面输入框下看到即时提示。
